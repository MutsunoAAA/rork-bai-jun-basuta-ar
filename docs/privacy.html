プライバシーポリシー

アプリ名: ばい菌バスターAR
最終更新日: 2026年4月22日

本プライバシーポリシーは、「ばい菌バスターAR」（以下「本アプリ」といいます）における、利用者の情報の取り扱いについて定めるものです。本アプリは子どもの歯みがきを楽しくするためのARアプリです。

1. 取得する情報

本アプリは、以下の情報のみをデバイス内で一時的に利用します。

  • カメラ映像: AR表示のためにカメラ映像をリアルタイムで画面に表示します。
  • 顔のランドマーク情報（TrueDepth API / ARKit Face Tracking）: 口が開いているか、笑っているかを判定するためだけに利用します。
  • 歯みがきセッション回数: キャラクター出現ロジックのため、端末内（UserDefaults）にのみ保存します。

2. 顔データ（Face Data）の取り扱い

本アプリは、顔データを一切保存・送信・第三者提供しません。

  • 収集する顔データ: ARKitが提供する「口の開閉度（jawOpen）」「笑顔度（mouthSmile）」等のブレンドシェイプ数値、および顔の位置情報のみを使用します。顔の画像、顔の3Dメッシュ、顔認証用の生体情報は一切収集・保存しません。
  • 利用目的: 口を開けた・笑ったことを検知し、口元にばい菌キャラクターを表示する演出のためだけに利用します。
  • 保存場所: 顔データはデバイスのメモリ上でのみ処理され、ストレージには保存しません。アプリを閉じると即座に破棄されます。
  • 第三者提供: 顔データを第三者（サーバー、解析サービス、広告ネットワーク等）に送信・共有することは一切ありません。
  • 保持期間: リアルタイム処理のみで、保持期間は実質ゼロ（アプリ実行中のメモリ上のみ）です。

3. カメラ映像の取り扱い

カメラ映像は画面に表示するためだけに利用し、録画・保存・外部送信は一切行いません。

4. 第三者への提供・第三者サービス

  • 本アプリは、第三者の解析サービス（Google Analytics等）を利用していません。
  • 本アプリは、第三者の広告ネットワークを利用していません。
  • 本アプリは、利用者の個人情報・デバイス情報・顔データを、いかなる第三者にも提供しません。

5. 子どものプライバシー

本アプリは子どもの利用を想定しています。個人を特定できる情報の収集、アカウント登録、外部通信、広告表示、アプリ内課金は行いません。

6. 権限について

  • カメラ: ARによるばい菌表示のため利用します。利用者はいつでもiOSの設定から権限を取り消すことができます。

7. 本ポリシーの変更

本ポリシーは必要に応じて更新されます。更新時はこのページにて告知します。

Privacy Policy (English)

App Name: Baikin Buster AR
Last updated: April 22, 2026

1. Information We Collect

The app processes the following information locally on the device only:

  • Camera feed — displayed in real time for the AR experience.
  • Face landmarks (TrueDepth API / ARKit Face Tracking) — used solely to detect whether the mouth is open or smiling.
  • Brushing session count — stored only locally via UserDefaults.

2. Face Data Handling

The app never stores, transmits, or shares face data with any third party.

  • What face data we use: Only ARKit blend shape values (e.g. jawOpen, mouthSmileLeft/Right) and face position are used. We do not capture facial images, 3D face meshes, or any biometric identifiers.
  • Purpose: To detect mouth-open and smile events so cartoon germ characters can appear around the mouth.
  • Storage: Face data exists only in volatile memory during the AR session and is discarded immediately when the session ends. Nothing is written to disk.
  • Third-party sharing: None. Face data is never sent off the device.
  • Retention: Effectively zero — in-memory, real-time processing only.

3. Camera Feed

The camera feed is only displayed on screen. It is never recorded, saved, or transmitted.

4. Third Parties

  • No third-party analytics are used.
  • No third-party advertising is used.
  • No user data is shared with any third party.

5. Children's Privacy

This app is designed for children. It does not collect personally identifiable information, require accounts, make network requests, show ads, or contain in-app purchases.
