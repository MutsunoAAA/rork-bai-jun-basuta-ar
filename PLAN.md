# ばい菌バスターARの子ども向けAR歯みがきアプリを作る

**Features**

- [x] 画面中央の大きなスタートボタンから、すぐに遊び感覚で始められます。
- [x] 口をしっかり開けた時だけ、口元の近くに可愛いばい菌たちがふわっと現れます。
- [x] 「にー」と笑った時にも、口元にばい菌が現れるようにします。
- [x] 口を閉じると、ばい菌たちはすぐに消えて、動きがわかりやすくなります。
- [x] ばい菌は数種類の色違いでランダムに出てきて、少し上下にゆれる楽しい動きを見せます。
- [x] ばい菌は小さめサイズに調整し、本番用に差し替えやすい見た目に整えます。
- [x] 歯ブラシや手でシャカシャカした動きを、より敏感な往復モーション判定で検知し、ばい菌が逃げるように消える演出を追加します。
- [x] 口を開いた瞬間に「ポヨヨン」、終わった時に「キラキラ」の音が鳴ります。
- [x] 下の大きな完了ボタンを押すと、ばい菌が消えて達成感のあるごほうび演出が出ます。
- [x] iPhoneでもiPadでも、同じ使い方で迷わず操作できます。
- [x] 将来、ばい菌イラストを本番用のオリジナル素材に差し替えやすい形にします。
- [x] 飽きない要素：歯磨き回数に応じてランダムでうさぎ・犬・猫・おにぎり・パン・星・りんごなどのキャラクターが登場します。
- [x] 最初の2回はばい菌のみ、3回目以降は40%の確率でばい菌、60%の確率でサプライズキャラクターが出現します。
- [x] セッション回数はUserDefaultsで永続化し、キャラクターごとに固有の登場・逃走メッセージを表示します。

**Design**

- [x] 全体はポップで明るく、幼児が怖がらないやさしい色づかいにします。
- [x] スタート画面は余計な情報を減らし、大きなボタンを主役にした安心感のある見た目にします。
- [x] カメラ画面は映像を大きく見せつつ、操作は下側にまとめて親が片手で押しやすくします。
- [x] ばい菌は間抜けで可愛い印象にして、歯みがきのイヤな気持ちを遊びに変える雰囲気にします。
- [x] 完了演出はキラキラ感を強めて、「できた！」が伝わる華やかな仕上がりにします。
- [x] 文字は少なめにして、子どもにも保護者にも一目で伝わる画面構成にします。
- [x] ARカメラ画面の上部説明や開き/にーの数値表示はなくし、顔が大きく映る見た目にします。
- [x] ばい菌は口元の上側だけでなく少し下側にも広がるように配置します。
- [x] シャカシャカ時は、ばい菌が慌てて逃げるのが伝わる勢いある動きにします。

**Pages / Screens**

- [x] スタート画面：大きなスタートボタンと、楽しい雰囲気が伝わる導入画面です。
- [x] ARカメラ画面：口を開けるとばい菌が現れ、下の完了ボタンでしめくくれるメイン画面です。
- [x] 完了画面：ばい菌が消え、キラキラ演出と「よくがんばったね！」で終われるごほうび画面です。

**App Icon**

- [x] 明るい背景に、にこっとした可愛いばい菌と歯のモチーフを組み合わせた、子ども向けで親しみやすいアイコンを作ります。
